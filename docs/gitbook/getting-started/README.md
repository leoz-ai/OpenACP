# Getting Started

Welcome to OpenACP! OpenACP lets you chat with AI coding agents — like Claude Code, Gemini CLI, or Copilot — directly from Telegram, Discord, or Slack. No browser tab switching, no terminal babysitting.

Pick the path that fits you best:

- **[What is OpenACP?](what-is-openacp.md)** — New here? Start with the big picture.
- **[Quick Start for Users](for-users.md)** — Someone else set up OpenACP and you just want to start chatting with an AI agent.
- **[Quick Start for Developers](for-developers.md)** — You want to install OpenACP on your machine and run it yourself.
- **[Quick Start for Contributors](for-contributors.md)** — You want to hack on OpenACP, add a feature, or fix a bug.
